File Toolbar

The File Toolbar provides a quick access to actions of the File Menu. The File Toolbar is visible when the checkbox in the  View->File Toolbar  menu is checked.

File Toolbar


Ctrl+N opens the New Model Dialog Box.
Ctrl+O opens the "Open Model File" dialog.
Reloads the model from disk. NOTE: A message box shows in case there are any edits to the model that could be lost
Ctrl+S saves the current model. Disabled if the model file is up to date.

Edit Toolbar

The Edit Toolbar provides a quick access to actions of the Edit Menu. The Edit Toolbar is visible when the checkbox in the  View->Edit Toolbar  menu is checked.

Edit Toolbar


locks the model to prevent any editing. You can still explore the model and generate code.
unlocks the model to allow editing.
Ctrl+Z performs undo of the last model edit (see Undo/Redo)
Ctrl+Y performs redo of the last model edit (see Undo/Redo)
Ctrl+X deletes the selected text or model item and places it in the Clipboard.
Ctrl+C copies the selected text or model item into the Clipboard
copies the selected elements of the current diagram to the Clipboard in the Scalable Vector Graphics (SVG) format. If no elements are selected the whole diagram is saved in the Clipboard. Disabled if no diagram is currently active.
Ctrl+V pastes the content of the Clipboard to the current model editor (file, diagram, or property sheet). Disabled if clipboard has no data or the data is in a format incompatible with the current view..
F5 selects a new Current Item based on the model-reference stored in the Clipboard (see also Model Link Comments).

View Toolbar

The View Toolbar provides a quick access to actions of the View Menu. The View Toolbar is visible when the checkbox in the  View->View Toolbar  menu is checked.

View Toolbar


Ctrl-E Toggles between show/hide the Model Explorer
Ctrl-M Toggles between show/hide the Model Search
Ctrl-P Toggles between show/hide the Property Editor
Ctrl-B Toggles between show/hide the Bird's Eye View
Ctrl-L Toggles between show/hide the Log Console
Ctrl-T Toggles between show/hide the Diagram Toolbox
Drop down box with four choices of grid for diagrams. Disabled if no diagram is currently active.

Tools Toolbar

The Tools Toolbar provides a quick access to actions of the Tools Menu. The part of the Tools Toolbar with keyboard shortcuts F8-F12 is customizable by means of the Manage Tools Dialog Box, so it might look differently in your system. The Tools Toolbar is visible when the checkbox in the  View->Tools Toolbar  menu is checked.

Tools Toolbar (Example)


launches the Code Generation License Dialog Box.
F7 generates code for all file-templates in the model (see also Code Engineering)
F8 invokes an external tool #1 (see also Manage Tools Dialog Box)
F9 invokes an external tool #2 (see also Manage Tools Dialog Box)
F10 invokes an external tool #3 (see also Manage Tools Dialog Box)
F11 invokes an external tool #4 (see also Manage Tools Dialog Box)
F12 invokes an external tool #5 (see also Manage Tools Dialog Box)
Esc terminates the process that is running the current tool. NOTE: enabled only when an external tool is running.
launches the Manage Tools Dialog Box
